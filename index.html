Tabla Comparativa

Característica Función principal Tipo de lenguaje Sintaxis básica Ubicación en el documento Relación
HTML (HyperText Markup Language) Estructura y contenido de la página web Lenguaje de marcado < h1 >, < p >, < img >, < table >, < form > Dentro del archivo .html Es la base sobre la que CSS y Bootstrap aplican estilos y diseño. Sin HTML, no hay elementos que estilizar.
CSS (Cascading Style Sheets) Estilo y presentación visual de la página web Lenguaje de estilos color: red;, font-size: 16px;, margin: 10px; Dentro del archivo .css o en la sección < style > del HTML Se usa para personalizar la apariencia de los elementos HTML. Bootstrap está construido sobre CSS.
Framework que facilita el diseño responsivo y estilizado Biblioteca basada en CSS y JavaScript Combinación de clases CSS predefinidas y componentes que te permiten construir páginas web rápidas, responsivas y estilizadas sin tener que escribir mucho código desde cero. class="btn btn-primary", class="container" Se incluye mediante a la CDN o archivo local Bootstrap usa clases CSS que se aplican a elementos HTML para darles estilo y funcionalidad sin escribir mucho código.
